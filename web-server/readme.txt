Для трансляции сервера прописываем в терминале команду "g++ server.cpp -o server"
Для запуска сервера прописываем в терминале команду "valgrind ./server"

Для трансляции клиента прописываем в терминале команду "g++ client.cpp -o client"
Для запуска клиента прописываем в терминале команду "valgrind ./client"
